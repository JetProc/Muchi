# MUMU

좋아했던 음악을 곡별 태그와 기억으로 보관하고, 여러 큐브가 이어진 나만의 음악 세계로 만드는 개인 아카이빙 서비스입니다.

## 핵심 경험

- iTunes 검색으로 곡을 빠르게 포착하고 30초 미리듣기
- 분류를 미뤄둘 수 있는 임시 보관함
- 시기·장소·감정에 따라 큐브 생성 및 관리
- 같은 곡도 큐브마다 독립적인 다중 태그, 성격, 시기, 장소, 사람, 메모 기록
- 곡·태그·메모를 함께 찾는 개인 아카이브 검색
- 저장한 시기와 계절을 바탕으로 한 음악 회고
- 캐릭터가 큐브 사이를 이동하는 2.5D 음악 세계
- JSON 백업·복원, 모션 감소, 오프라인 앱 셸

현재 버전은 수요 검증용 로컬 우선 MVP입니다. 기록은 계정이나 서버가 아니라 사용 중인 브라우저의 `localStorage`에만 저장됩니다.

## 실행

Node.js `22.13.0` 이상이 필요합니다.

```bash
npm install
npm run dev
```

## 검증

```bash
npm run lint
npm run typecheck
npm test
```

`npm test`는 프로덕션 빌드, 주요 화면의 서버 렌더링, PWA 자산, 큐브별 곡 맥락의 독립성을 확인합니다.
